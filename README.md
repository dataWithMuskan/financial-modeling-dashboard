# 📊 Financial Modeling Dashboard

## 🧾 Project Overview

This project analyzes the financial performance of a retail business (2021–2024) using Microsoft Excel.
It includes revenue, cost, and profit analysis along with break-even and scenario analysis.

---

## 📈 Key Features

• Revenue vs Cost Analysis
• Profit Trend Visualization
• Break-even Analysis
• Scenario Analysis (Best, Normal, Worst)
• Waterfall Chart for cost breakdown

---

## 🧠 Key Insights

• Profit increases after break-even (~714 units)
• Selling price has strong impact on profit
• Cost structure significantly affects profitability
• Business performance improves over time

---

## 💼 Business Recommendations

• Increase sales volume through marketing
• Optimize pricing strategy
• Reduce variable costs
• Maintain operations above break-even
